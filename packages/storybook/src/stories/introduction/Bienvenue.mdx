# Bienvenue

Ce Design System est un projet personnel open-source, conçu pour être portable, maintenable et indépendant de tout contexte professionnel.

Il repose sur des principes simples : **Figma comme source de vérité**, des tokens stables, des composants accessibles et une documentation en français.

---

## Stack technique

- **Monorepo** — pnpm + Turborepo
- **Composants** — React + TypeScript strict
- **Styles** — CSS Modules + CSS Variables
- **Tokens** — Style Dictionary v4
- **Tests** — Vitest + Testing Library + jest-axe
- **Documentation** — Storybook 8
