---

## Conventions

- **Langue** — documentation et commentaires en français
- **Branche** — toujours travailler sur `main` directement
- **Commits** — format `type(scope): description` en français
- **Accessibilité** — WCAG 2.1 AA obligatoire sur chaque composant
- **Tests** — couverture minimum 80%

---

## Ajouter un composant

1. Créer le composant dans `packages/react/src/components/NomComposant/`
2. Exporter depuis `packages/react/src/index.ts`
3. Créer le dossier de stories dans `packages/storybook/src/stories/components/NomComposant/`
4. Ajouter les 5 fichiers : `Docs.mdx`, `playground.stories.tsx`, `variants.stories.tsx`, `states.stories.tsx`, `usage.mdx`
5. Mettre à jour le `storySort` dans `.storybook/preview.ts`
