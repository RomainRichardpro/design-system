## Accessibilité

| Critère | Détail |
|---|---|
| **Rôle ARIA** | `<button>` natif — rôle `button` implicite |
| **État chargement** | `aria-busy="true"` + label masqué visuellement via `.srOnly` |
| **État désactivé** | Attribut natif `disabled` — exclu du focus, non interactif |
| **Focus visible** | Halo `:focus-visible` — `var(--focus)` · `var(--border-width-03)` · offset 4px |
| **Navigation clavier** | `Tab` pour atteindre · `Enter` ou `Space` pour déclencher |
| **Contraste** | Testé WCAG 2.1 AA sur `primary` et `secondary` |
